Clientes and Servers

Computers connected to the Web are called Clientes and servers.

  • Clientes are the typical web user's internet-connected devices(for example, your computer connected to your Wi-Fi, or your ohone connected to your mobile network) and web-accessing software available on those devices (usually a web browser like Firefoc or Chrome)
  • Servers are computers that sore webpages, sites op apps.When a client device want to access a webpage, a copy of the webpage is downloades from the server onto the client machine to be displayed in the user's web browser.

So What Actually Happens?

  • 1.The browser goes to the DNS server and finds the real address of the server that the website lives on.
  • 2.The browser send an HTTP request message to the server asking it to send a copy of the website to the client.
  • 3.If the server approves the client's request, the server send the client a "200 OK" message, which means "permission grated! here you go!", and the stars sending the website's file to the browser as a series of small chunks.
  • 4.The browser assembles the small chunks into a complete website and displays it to you!
